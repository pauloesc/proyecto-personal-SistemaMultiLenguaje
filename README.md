# proyecto-personal-SistemaMultiLenguaje
Proyecto personal.
El sistema tiene como objetivo poder cargar un documento estructurado en forma de capitulos y parafos para luego poder hacer asignaciones de esos parrafo a traductores los cuales se encargaran de traducir dichos parafos.
Los traductores traducen a idiomas especificos (los idiomas que saben).
Los usuarios pueden denunciar malas traducciones.
